# 個人作品集網站專案

## 專案目標
創建一個簡約風格的個人作品集網站，適合以下對象使用：
- 設計師
- 開發者
- 攝影師
- 創作者

## 核心特色
- One Dark Pro 暗色主題設計
- 護眼友好的配色方案
- 響應式佈局（支援桌面、平板、手機）
- 大圖展示效果
- 自我介紹區塊
- 互動式聯絡表單

## 技術棧
- HTML5
- CSS3（採用 CSS 變數實現主題配色）
- JavaScript
- 外部庫：
  - Font Awesome (CDN) - 用於圖示
  - AOS (CDN) - 用於滾動動畫效果

## 主題配色
使用 One Dark Pro 主題配色方案：
```css
--primary-color: #282c34    /* 主要背景色 */
--secondary-color: #61afef  /* 強調色，用於連結和按鈕 */
--text-color: #abb2bf      /* 主要文字顏色 */
--light-color: #21252b     /* 次要背景色 */
--accent-color: #98c379    /* 次要強調色 */
--border-color: #3e4451    /* 邊框顏色 */
--hover-color: #3e4451     /* 懸停效果顏色 */
```

## CSS 設計原則
1. 使用 CSS 變數管理顏色主題
2. 保持良好的顏色對比度以確保可讀性
3. 統一的過渡動畫效果（300ms ease）
4. 響應式設計斷點：
   - 手機版：max-width: 480px
   - 平板版：max-width: 768px

## 開發指南

### 項目設置
1. 本項目不需要特別的建置步驟，直接在瀏覽器中打開 `index.html` 即可運行
2. 所有依賴都使用 CDN 載入，確保穩定的連線環境

### 代碼結構
- `index.html`：主要的 HTML 結構
- `css/style.css`：所有樣式定義，使用 CSS 變數管理主題
- `js/main.js`：所有 JavaScript 功能實現

### 主要功能模組
1. **主題管理**
   - 檔案：`js/main.js`
   - 相關函數：`themeToggle` 事件監聽器
   - 儲存位置：LocalStorage 的 'theme' 鍵

2. **表單驗證**
   - 檔案：`js/main.js`
   - 主要函數：`validateEmail`, `showError`, `removeError`
   - 觸發事件：表單提交和輸入變更

3. **作品集過濾**
   - 檔案：`js/main.js`
   - 相關變數：`filterButtons`, `workItems`
   - CSS 類別：`show`, `hide`

4. **動畫效果**
   - 使用 AOS 庫
   - 配置在 `DOMContentLoaded` 事件中
   - 手機版預設停用

### 擴展指南
1. **添加新作品**
   - 在 `works-grid` 中添加新的 `work-item`
   - 確保設置正確的 `data-category`
   - 使用 `loading="lazy"` 優化載入

2. **添加新主題**
   - 在 `:root` 中定義新的顏色變數
   - 在 `[data-theme="新主題"]` 中覆寫變數

3. **添加新功能**
   - 保持模組化設計
   - 遵循現有的命名規範
   - 注意裝置相容性

### 注意事項
1. 在修改主題時需同時更新淺色和深色模式
2. 新增功能時要考慮響應式設計
3. 保持程式碼註釋的一致性
4. 確保跨瀏覽器兼容性
